# Rate My Mentor

A verifiable, censorship-resistant feedback layer for workplace experiences.

---

What is this?

Rate My Mentor is a protocol for **verifiable workplace feedback**.

Users:
- prove real employment experience (credential / attestation)
- submit structured reviews on company / team / mentor

System:
- ensures credibility via attestation + anti-sybil
- stores proof on-chain (hash, signature, timestamp)
- stores sensitive content off-chain (encrypted)

Goal:
> build a trustable, non-manipulable reputation layer for workplace experience
